轉型正義教育黑客松

促進轉型正義委員會委託《沃草》於 11/23 舉辦「轉型正義教育黑客松」，聚集中學教師對轉型正義的關注，向社會各界徵集提案，運用現有素材，設計適合學生的行動方案，推動轉型正義普及化。

什麼是黑客松？為什麼是黑客松？

《沃草》團隊承促轉會委託，辦理「轉型正義教育黑客松」。對我們來說，「黑客松」的精神就是「發現問題、動手做事、互相幫助」。「轉型正義」、「教育」同時是專業，也是眾人之事，需要來自各種背景、各種專業的朋友共同參與，才能更瞭解問題、找出解方。我們想鼓勵提案者，不要侷限自己，儘量提出你想做的事，提案沒有很完整也沒有關係，也許就會在黑客松找到志同道合的夥伴。我們也想提醒參與者，保持開放的心態，因為黑客松協作的過程，也是互相傾聽、一起前進的過程。

在整個黑客松的過程中，不管是活動前的網路公開討論，或是活動現場，提案者、參與者能與不同領域的朋友交流、腦力激盪、串聯、協力開發促進轉型正義的各種行動方案。我們期許，此次活動的成果，能成為所有教育工作者的重要資源，為轉型正義教育的跨域串連、落實，創造更多可能性，進一步推動轉型正義的普及化。
